# CLAUDE.md

## Project Overview

A comprehensive Markdown wiki for modern Angular development — covering Angular 17+ patterns including standalone components, signals, new control flow, SSR with hydration, and zoneless change detection.

## Repository Structure

This is a documentation repository — pure Markdown files with Mermaid diagrams, no build step.

- `README.md` — Learning roadmap with Mermaid graph
- Individual topic files at root level

## Content Guidelines

- Focus on Angular 17+ patterns. Reference legacy (NgModules, `*ngIf`, Zone.js) only for migration context.
- Use Mermaid diagrams for complex flows and architecture
- Cross-link related concepts between pages
- Include practical code examples

## No Build Required

This is a static Markdown wiki. Edit `.md` files directly.
